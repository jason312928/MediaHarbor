# Security Policy

Please do not disclose a vulnerability in a public issue. Contact the repository maintainer privately through the security-reporting method configured on the GitHub repository.

Never include browser cookies, authentication headers, private media URLs, downloaded files, or other personal data in a report. A useful report includes the affected commit, macOS version, reproduction steps, impact, and a minimal sanitized proof of concept.

MediaHarbor executes the official yt-dlp binary and optional FFmpeg tools. Keep them updated, and only use browser-cookie integration with a profile whose data you are comfortable exposing to yt-dlp.
